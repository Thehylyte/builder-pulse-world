## 🚀 Quick Start (3 Steps)

### 1. Build the App

```bash
npm run build:client
npm run mobile:sync
```

### 2. Open in Xcode

```bash
npm run mobile:ios
```

### 3. Follow the Checklist

Open `DEPLOYMENT_CHECKLIST.md` and follow each step to submit to App Store.
